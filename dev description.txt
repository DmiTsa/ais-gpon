Система учета и инвентаризации компонентов оптических сетей GPON (СУИ GPON)
--------------------
язык javaScript, web интерфейс, клиент серверное приложение с хранением в БД
клиент web (React), сервер NodeJs, БД postgreSql
--------------------
программа для хранения информации об участках сети - кабель с оконечными (промежуточными) устройствами.
Элементарная еденица (сущность) хранения - КУ (кабельный участок). два устройства ОКУ (оконечное кабельное устройство) соединенные кабелем. ОКУ независимые единицы, КУ входят в состав СМ (станционная магистраль*)

----
сущность КУ (№, марка кабеля, длина по КСГ (канализация, стена, грунт))
сущность СМ (Имеет №, набор КУ, паспорт)
сущность Инв№ (набор КУ)
каждый КУ должен иметь принадлежность к Инв.№ и к СМ
сущность ОКУ (№, описание, адрес) в перспективе - состав по более мелкому оборудованию
каждый ОКУ должен иметь принадлежность к КУ и к инв.№


---------------------------------------------------------------------------------------------------------
Станционная магистраль - вид ПОМ (пассивная оптическая магистраль) соединенной со станцией. Несколько соединенных КУ, имеющих общий номер на стании.
